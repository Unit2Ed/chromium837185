Test Case for Chromium bug 837185

  1. Trigger Protocol and notice that Chrome does nothing at all
  2. Register an external protocol, eg chromiumtest10:// and point it at notepad.exe
    • Download and run pre-prepared register.reg file,
    • or follow instructions at docs.microsoft.com to create the protocol by manually editing the registry
  3. Trigger Protocol again (identical to step 1)
  4. Check "Open these types of links in the associated app"
  5. Click "Launch" and notice that Notepad opens
  6. Unregister the protocol
    • Download and run pre-prepared unregister.reg file,
    • or delete the registry keys you manually created following the Microsoft guide
  7. Trigger Protocol again (identical to step 1)
